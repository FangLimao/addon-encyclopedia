    Block Renderer

    Minecraft Block Renderer工具可以根据方块的纹理将其渲染成其标准视图。

    Minecraft Block Renderer

    使用方式

    1. 打开网页，选择方块的模型1
    2. 选择导出图片的颜色2
    3. 选择方块的纹理，你可以选择原版纹理也可以自己上传纹理3
    4. 选择导出图片的分辨率，点击Save Result获取导出图片

    注释

    1. 即 Model 选栏 ↩

    2. 即 Color 选栏 ↩

    3. 即 Top Side, Left Side, Right Side 选栏 ↩
