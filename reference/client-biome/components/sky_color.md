# 客户端群系文件文档 - sky_color

> [!INFO]
> 本文译自[Microsoft Learn](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可

`minecraft:sky_color`组件设置生物群系的天空颜色。

## 参数

| 名称 | 类型 |  可选  | 说明 |
|:-----------|:-----------|:-------|:-----------|
|sky_color| RGB值 | 是| 天空颜色的RGB值 |


## 范例

```json
"minecraft:sky_color": {
  "sky_color": "#000000"
}
```
