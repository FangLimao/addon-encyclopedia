# 物品组件 - minecraft:hover_text_color
> [!INFO]
> 本文译自[微软开发文档](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可

    
`minecraft:hover_text_color`组件决定鼠标悬停在物品时名称的颜色。

## 参数
| 名称 | 默认值 | 类型 | 说明  |
|:----------|:----------|:----------|:----------|
| value | *未设置* | 字符串 | 鼠标悬停时物品名字的颜色 |
