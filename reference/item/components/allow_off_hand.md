# 物品组件 - minecraft:allow_off_hand
> [!INFO]
> 本文译自[Microsoft Learn](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可


`minecraft:allow_off_hand`组件决定物品是否可以放在副手槽中。

## 参数
| 名称 | 默认值 | 类型 | 说明  |
|:----------|:----------|:----------|:----------|
| value | *未设置* | 布尔值 | 物品是否可以放在副手槽中 |
