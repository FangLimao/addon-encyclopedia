# 物品组件 - minecraft:should_despawn
> [!INFO]
> 本文译自[Microsoft Learn](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可

    
`minecraft:should_despawn`组件决定物品的实体形式是否可以超时消失。

## 参数

| 名称 | 默认值 | 类型 | 说明  |
|:----------|:----------|:----------|:----------|
| value | *未设置* | 布尔值 | 物品的实体形式是否可以超时消失 |
