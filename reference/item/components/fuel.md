# 物品组件 - minecraft:fuel
> [!INFO]
> 本文译自[微软开发文档](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可

    
`minecraft:fuel`组件将物品设置为可以为熔炉供能的燃料。

## 参数
| 名称 | 默认值 | 类型 | 说明  |
|:----------|:----------|:----------|:----------|
| duration | *未设置* | 小数 | 为熔炉供能的时间，最小为0.05 |

## 范例
```json
"minecraft:fuel":{
    "duration": 3.0
}
```
