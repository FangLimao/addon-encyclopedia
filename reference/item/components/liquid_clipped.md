# 物品组件 - minecraft:liquid_clipped
> [!INFO]
> 本文译自[Microsoft Learn](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可

    
`minecraft:liquid_clipped`组件决定物品是否可以与流体交互。

## 参数
| 名称 | 默认值 | 类型 | 说明  |
|:----------|:----------|:----------|:----------|
| value | *未设置* | 布尔值 | 定义物品是否可以与流体交互 |
