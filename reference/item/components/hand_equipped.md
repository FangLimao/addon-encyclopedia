# 物品组件 - minecraft:hand_equipped
> [!INFO]
> 本文译自[微软开发文档](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可

    
`minecraft:hand_equipped`组件决定物品在手中是否以工具形式握持。

## 参数

| 名称 | 默认值 | 类型 | 说明  |
|:----------|:----------|:----------|:----------|
| value | *未设置* | 布尔值 | 决定物品在手中是否以工具形式握持 |
