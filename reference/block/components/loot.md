# 方块组件 - minecraft:loot

> [!INFO]
> 本文译自[Microsoft Learn](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可

`minecraft:loot`组件设置方块的掉落物。

| minecraft:loot | 组件信息 |
| ----------------------- | -------- |
| 数据类型                |     字符串     |
| 默认值                  |          |

可用的值是行为包战利品表的文件路径，最多256个字符。

## 范例

```json
"minecraft:loot": "beh/loot_tables/custom_block.json"
```
