# 方块标签
本文档展示了所有原版方块标签，你可以使用`minecraft:tags`组件把这些标签添加到自定义方块中。

你可以为自定义方块添加自己的标签，但是只有原版标签可以使用`minecraft`命名空间。

!!! note "注意"

    早期的方块标签没有命名空间
