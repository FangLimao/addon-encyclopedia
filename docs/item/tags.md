!!! note "注意"

    本段所述标签基于 1.19.50 的游戏内容

## 自定义物品标签
你可以使用 [minecraft:tags](components/tags.md) 组件来设置一个自定义物品的标签。

[^1]: 原文为“Bookshelf Books”
[^2]: 原文为“Logs that Burn”
[^3]: 应用于灵魂火、灵魂火把和灵魂灯笼，猪灵会躲避应用该标签的物品和方块
