# 物品组件 - minecraft:use_animation
!!! info "版权说明"
    本文译自[微软开发文档](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可
    
`minecraft:use_animation`组件定义物品使用的动画。

## 参数

| 名称 | 默认值 | 类型 | 说明  |
|:----------|:----------|:----------|:----------|
| value | | 字符串 |  物品使用的动画 |

## 可用动画
- eat
- drink
- bow
- block
- camera
- crossbow
- none
- brush
- spear
- spyglass
