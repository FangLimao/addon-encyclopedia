# 物品组件 - minecraft:glint
!!! info "版权说明"
    本文译自[微软开发文档](https://learn.microsoft.com/en-us/minecraft/creator/)，按照 CC BY 4.0 协议进行许可
    
`minecraft:glint`组件决定物品是否有附魔光效。

## 参数

| 名称 | 默认值 | 类型 | 说明  |
|:----------|:----------|:----------|:----------|
| value| *未设置* | 布尔值 | 决定物品是否有附魔光效 |

## 范例
```json
"minecraft:glint": false
```
